# Consul ECS 0.5.x

## Release Highlights

- **Audit Logging (Enterprise) :** This feature supports audit logging by providing an `audit-logging` flag, providing a log of authenticated events if enabled.

- **Support Consul AWS IAM auth method :** This feature provides support obtaining ACL tokens from Consul's AWS IAM auth method.

- **Mesh Gateways :** This feature introduces Consul WAN federation via mesh gateways ECS tasks, allowing for Consul servers in different datacenters to be federated.

## Supported Software Versions

- Consul: 1.12.x
