# Compatibility

## Consul

Below are the supported Consul versions with compatible Consul-Terraform-Sync (CTS) versions. The latest CTS binary targets supporting the latest patch version of the three most recent Consul minor versions.

| Consul Version               | Compatible CTS Version |
| ---------------------------- | ---------------------------------------- |
| 1.8+                         | 0.1+                                     |

## Terraform

CTS is compatible with the following Terraform OSS versions:

| CTS | Compatible Terraform Version |
| --------------------- | ---------------------------- |
| 0.2+                  | 0.13 - 1.0                   |
| 0.1                   | 0.13 - 0.14                  |

## Terraform Cloud

CTS integration with Terraform Cloud is supported for the following:

| CTS Enterprise | Terraform Cloud            | Version            |
| -------------------------------- | -------------------------- | ------------------ |
| 0.4+                             | Terraform Cloud            | Latest             |
| 0.3+                             | Terraform Enterprise       | v202010-2 - Latest |
