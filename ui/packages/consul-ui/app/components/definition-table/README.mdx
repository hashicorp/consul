# definition-table

Simple CSS component to render a `dl` similar to a table with column headers.

```hbs preview-template
<div class="definition-table">
  <dl>
    <dt>Title 1</dt>
    <dd>Value</dd>
    <dt>Title 2</dt>
    <dd>
      <CopyableCode
        @name="Title 2"
        @value="Value"
      />
    </dd>
  </dl>
</div>
```


```css
.definition-table {
  @extend %definition-table;
}
```
